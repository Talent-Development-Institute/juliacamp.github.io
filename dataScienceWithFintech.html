Julia Programming Fundamentals

4 Lessons / 12 Week

Summer 2020 is unlike any other summer in modern times.

Course Description

Schedule: MWF 9:30 - 11:00

This module introduces Julia from MIT, a high-level, high-performance, dynamic programming language that is optimized for scientific and numerical computing. While it is a general purpose language and can be used to write any application, many of its features are well-suited for high-performance numerical analysis and computational science. In this module, we will focus on Julia programming skills and how to develop data science applications based on Julia.

Learning Outcomes

Upon completion of this module, the student shall be able to:

  • Solve problems using systematic approaches
  • Identify and develop problem solutions related to data science with Julia and packages
  • Visualize the model developed by Julia with Julia package

Sessions

The Julia Programming is a 12.5-hour program delivered over the course of one week, and includes 2.5 hours of tutorial and an optional exam.
