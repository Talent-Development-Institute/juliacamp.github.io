Our Program

JuliaCamp is a program that attracts students with high aptitudes for curiosity, excellence and resilience, and prepares them to take on social and economic challenges by leveraging data science techniques and using the Julia programming language. This is a self-based program that can be completed between 1-4 weeks.

Courses

JuliaCamp has a main module for Julia and two stand-alone modules for those interested in the science of disease modelling and mechanics of making money on the financial markets

5 Sessions / 1 Tutorial

Julia Programming Fundamentals

JUL teaches Julia, a new programming language that is optimized for scientific and numerical computing

5 Sessions / 1 Tutorial

Data Science with Epidemiology

EPI teaches the mathematics of data science via disease modelling, specifically with the COVID-19 pandemic

5 Sessions / 1 Tutorial

Data Science with Fintech

FIN teaches the mathematics of data science for financial engineering, a sought-after skill in investment banking

About Julia Camp

JuliaCamp is a week-long program for high school students to learn practical data science skills from the inventors of Julia, the gold standard for programming in data science. No prior experience necessary!
